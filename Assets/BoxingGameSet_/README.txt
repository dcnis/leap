Boxing Game Set

Assets
The package contains 16 different prefabs (BoxingGameSet/Prefabs) and 3 different
rigged mecanim ready characters (BoxingGameSet/Prefabs).
Demo scene included (BoxingGameSet/ExampleScene).

Animations
The package only contains some base animations (BoxingGameSet/Animations). Combine these
with other mecanim animations (use masks, layers and mirror functions) for the good result
in your game.
